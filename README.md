# E-Commerce Sales Analysis

## Project Overview
Analyzed 50,000+ e-commerce transactions to uncover sales trends, top products, and revenue insights.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- Excel

## Key Findings
- Top 3 products contributed 68% of total revenue
- United Kingdom generated highest revenue: 219,785
- November had peak sales month

## Dataset
Original dataset downloaded from Kaggle:
https://www.kaggle.com/datasets/carrie1/ecommerce-data

## Files
- ECommerce_Analysis.ipynb - Main Python notebook
- top_products.png - Top 10 products chart
- monthly_trend.png - Monthly revenue trend
- country_revenue.csv - Country wise revenue data
